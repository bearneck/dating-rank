## 💡 自定义建议

### 修改评分规则
编辑 `SCORING` 对象：
```javascript
const SCORING = {
    male: {
        height: {150:0, 155:0, 160:0, 165:5, 170:10, 175:14, 180:17, 185:20, 190:20},
        // 修改对应分数...
    }
}
```

### 调整等级阈值
编辑 `GRADES` 数组：
```javascript
const GRADES = [
    {min: 90, name: '超夯', emoji: '🔥🔥🔥', rank: 98, comment: '你的评语'},
    // 修改等级和评语...
]
```

### 添加新维度
1. 在 HTML 添加新问题
2. 在 `SCORING` 添加评分规则
3. 在 `calculateResult` 添加计算逻辑
